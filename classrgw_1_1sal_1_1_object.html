Object abstraction

This represents an Object. An Object is the basic unit of data storage. It represents a blob of data, a set of metadata (such as size, owner, ACLs, etc.) and a set of key/value attributes. Objects may be versioned. If a versioned object is written to, a new object with the same name but a different version is created, and the old version of the object is still accessible. If an unversioned object is written to, it is replaced, and the old data is not accessible.
