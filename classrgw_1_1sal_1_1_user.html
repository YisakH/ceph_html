User abstraction

This represents a user. In general, there will be a User associated with an OP (the user performing the OP), and potentially several others acting as owners. Lifetime of a User is a bit tricky , since it must last as long as any Buckets associated with it. A User has associated metadata, including a set of key/value attributes, and statistics (including usage) about the User.
